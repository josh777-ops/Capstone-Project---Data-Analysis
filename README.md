# Capstone-Project---Data-Analysis

# 🚗 Automobile Data Analysis – Exploratory Data Analysis (EDA)

## Overview

This project is an Exploratory Data Analysis (EDA) of an automobile dataset completed as part of the HyperionDev Data Analysis Capstone Project. The aim of the project was to clean, prepare, analyse, and visualise the data to uncover meaningful insights about different automobile characteristics.

The analysis was completed using Python in a Jupyter Notebook, following industry-standard data cleaning and exploratory data analysis techniques.

---

## Objectives

The project focuses on:

* Loading and exploring a real-world dataset
* Cleaning and preparing the data for analysis
* Identifying and removing duplicate records
* Handling missing values
* Converting columns to appropriate data types
* Performing exploratory data analysis
* Creating informative visualisations
* Drawing conclusions from the data

---

## Dataset

The project uses the provided **automobile.txt** dataset containing information about various automobiles, including attributes such as:

* Price
* Horsepower
* Engine size
* Fuel type
* Number of doors
* Body style
* Drive wheels
* Fuel efficiency
* Vehicle dimensions
* Weight
* Manufacturer information

## Running the Project

Clone the repository:

```bash
git clone https://github.com/yourusername/automobile-eda.git
```

Navigate to the project folder:

```bash
cd automobile-eda
```

Launch Jupyter Notebook:

```bash

```

Open:

```text
automobiles.ipynb
```

Run all notebook cells to reproduce the analysis.
